# Terminal Shooting Game（终端射击游戏）

## 项目简介
这是一个基于 Python `curses` 库编写的终端射击小游戏。  
玩家位于地图中央，敌人会从地图边缘生成并自动向玩家移动。  
玩家通过方向键（W/A/S/D）射击敌人，尽可能获得更高分数。  
游戏支持**历史最高分持久化保存**，并可手动清零最高分。

---

## 如何运行

### 环境要求
- Python 3.x
- macOS / Linux  
- Windows 用户需使用支持 `curses` 的环境（如 WSL）

### 运行步骤
1. 克隆或下载本项目
2. 进入项目目录
3. 运行游戏：

```bash
python main.py
```

首次运行会自动生成 best_score.txt 用于保存历史最高分。

操作说明
主界面（MENU）

y ：开始游戏

n ：退出游戏

o ：清零历史最高分

游戏中（PLAYING）

W ：向上射击

A ：向左射击

S ：向下射击

D ：向右射击

敌人接触玩家会扣血，血量为 0 时游戏结束

游戏结束（GAME OVER）

e ：重新开始游戏

q ：退出游戏

o ：清零历史最高分
